#ENS410064
Este repositório contém os materiais desenvolvidos na disciplina de Python for Environmental Applications do Programa de Pós-Graduação em Engenharia Ambiental. 

A pasta está organizada da seguinte maneira:
- Notebook: pasta como os Jupyter notebook;
- .gitgnore: documento com os arquivos ignorados;
- Atividade_02: atividades para plotagem de gráficos utilizando os dados de coleta de RDO e RPU disponibilizado pelo SNIS no município de Florianópolis.
